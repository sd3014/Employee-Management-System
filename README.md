# Employee-Management-System
This project is an employee management system that stores employee data, allowing for insertion, editing, and deletion. It’s fully implemented using JSwing for the interface and MySQL for data storage.

📌 Employee Management System – Java GUI Application
A desktop-based Employee Management System built in Java using Swing/AWT for GUI. This application allows organizations to efficiently manage employee records with features like:

🔐 Secure Login System – Admin/user authentication before access.

➕ Add Employee – Store details such as name, designation, salary, and contact.

📝 Edit Employee – Update existing employee records easily.

❌ Remove Employee – Delete records securely from the database.

📄 View Employee – Search and display employee details in an organized table format.

💾 Data Storage – Employee records stored in a backend database (e.g., MySQL/SQLite).

🖥️ User-Friendly Interface – Built with Java Swing for smooth and intuitive navigation.

Tech Stack:

Language: Java

GUI Toolkit: Swing / AWT

Database: MySQL / SQLite

Architecture: MVC-based modular design
